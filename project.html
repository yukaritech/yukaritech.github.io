Project 1: 屋内3Dマップシステム Torch

Arduino ESP32を使用したIoT対応のスマートライトシステム。 スマートフォンから遠隔操作可能で、明るさや色を自由に調整できます。

Arduino IoT LED制御

開発期間: 2024年4月 - 2024年6月
参加メンバー: 3名

Project 2: アヒルAI

センサーを使った自動植物給水システム。 土壌の湿度を監視し、必要に応じて自動で給水を行います。

センサー 自動化 環境監視

開発期間: 2024年7月 - 2024年9月
参加メンバー: 2名

Project 3: ADHD向けデバイス開発 Project D

音楽に合わせてLEDが光るビジュアライザー。 周波数解析を行い、音楽のリズムや音程に応じた美しい光の演出を作り出します。

音響処理 LED Matrix Python

開発期間: 2024年10月 - 2024年12月
参加メンバー: 4名
